# 🚀 Fundamental Booster – User Information Collector

## 📌 Project Overview

### This project is a beginner-friendly Python application developed using Jupyter Notebook. It demonstrates basic programming concepts by collecting user information and displaying it in a structured and meaningful way.
### The program also provides additional technical details such as data types and memory references, making it useful for understanding core Python fundamentals.

## 🎯 Features

### • Accepts user input (Name, Age, Height, Favourite Number)
### • Performs type casting (int(), float(), str())
### • Displays user data in a formatted output
### • Shows data types using type()
### • Displays memory address using id()
### • Calculates approximate birth year

## ▶️ How It Works

### 1. The program starts with a welcome message.
### 2. It asks the user to enter basic details:
###       • Name
###       • Age
###       • Height
###       • Favourite Number
### 3. The input values are converted into appropriate data types.
### 4. The program stores and processes the data.
### 5. It displays:
###      • Entered information
###      • Data types
###      • Memory addresses
### 6. It calculates the approximate birth year using:

####          Birth Year = Current Year - Age
### 7. Finally, the program shows the result in a clean format.
